## 🧩 What FilzaJailedDS Does

FilzaJailedDS is a tweak that adds file access features to the Filza iPA for jailed iOS devices.

It is built for iOS 17.0 to 26.0.1.

Use it if you want to browse files inside Filza on a non-jailbroken device with a supported setup.

## ✅ What You Need

- A Windows PC
- A browser with downloads enabled
- The Filza iPA version 4.0.0 or earlier
- An iPhone or iPad on a supported iOS version
- A way to install the iPA on your device
- Enough free space on your PC and device
